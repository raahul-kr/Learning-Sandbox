# 🎮 Backend Dev Learning Games & Resources

> Level up your backend skills — one game at a time. Click any link to visit the resource.

## 🗄️ 1. SQL & Database Mastery

*Practice JOINs, subqueries, and database design through gameplay.*

| Resource | Level | Description | Link |
|----------|-------|-------------|------|
| 🔍 SQL Murder Mystery | Intermediate | You're a detective in SQL City. Query crime scene reports, interviews, and witness tables to find the killer. Excellent for complex JOINs and subqueries. | [Visit →](https://mystery.knightlab.com/) |
| 🏝️ SQL Island | Beginner | A browser-based adventure where you use SQL commands to interact with inhabitants and escape an island. Guided and great for absolute beginners. | [Visit →](https://sql-island.informatik.uni-kl.de/) |
| 🚀 Schemaverse | Advanced | A space-based strategy game where the entire world IS a PostgreSQL database. Control your fleet by writing raw SQL commands — deployed 24/7. | [Visit →](https://schemaverse.com/) |
| 📚 SQLZoo | Beginner | Interactive SQL tutorials with real exercises. Learn SELECT, JOIN, GROUP BY, and window functions through incremental challenges with instant feedback. | [Visit →](https://sqlzoo.net/) |
| ⚔️ SQL Practice | Intermediate | Competitive SQL challenges with leaderboards. Sharpen your skills under time pressure against other developers. | [Visit →](https://www.sql-practice.com/) |

| Resource | Level | Description | Link |
|----------|-------|-------------|------|

## 🔐 4. Security & Systems (Side Quests)

*Linux, networking, and security — the stuff servers are made of.*

| Resource | Level | Description | Link |
|----------|-------|-------------|------|
| 🐧 OverTheWire: Bandit | Beginner | A wargame that teaches Linux command-line basics level by level. Since backend code runs on servers, knowing your way around a terminal is non-negotiable. | [Visit →](https://overthewire.org/wargames/bandit/) |
| 🕵️ HackTheBox | Advanced | Hands-on cybersecurity labs with real machines to exploit. Teaches you what NOT to do in your APIs and servers. Widely used for security certifications. | [Visit →](https://www.hackthebox.com/) |
| 🌐 TryHackMe | Beginner | Guided cybersecurity learning paths in the browser. Covers networking, Linux, web security, and more. More beginner-friendly than HackTheBox. | [Visit →](https://tryhackme.com/) |
| 🧱 PicoCTF | Intermediate | Free cybersecurity competition with challenges year-round. Covers binary exploitation, reverse engineering, forensics, and web vulnerabilities. | [Visit →](https://picoctf.org/) |
